# Data

| File | Description |
| --- | --- |
| `snli_train_entailment_1k.jsonl` | 1,000 randomly sampled SNLI train entailment problems (seed 42) used for the experiments. One JSON object per line with `id`, `premise`, `hypothesis`, `gold_label`, `dataset`, `split`. Can be directly used with `agentic-pipeline-langpro --input`. |

Sampled from the [SNLI corpus](https://nlp.stanford.edu/projects/snli/) (CC BY-SA 4.0).
